# Eye Controlled Mouse with Voice Assistant

## Overview

This project enables users to control the mouse cursor using eye movements and perform actions using voice commands. It is built using Python, OpenCV, MediaPipe, SpeechRecognition, and PyAutoGUI for hands-free computer interaction.

## Features

- Eye-controlled cursor movement
- Blink detection for mouse clicks
- Voice assistant support
- Real-time webcam tracking
- Hands-free computer interaction

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- SpeechRecognition
- NumPy

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/dhanashri-kaisare/Eye_controlled_mouse_with_voice_assistant.git
```

### 2. Navigate to the project folder

```bash
cd Eye_controlled_mouse_with_voice_assistant
```

### 3. Create a virtual environment

```bash
python -m venv myenv
```

### 4. Activate the virtual environment

**Windows**

```bash
myenv\Scripts\activate
```

### 5. Install the required libraries

```bash
pip install -r requirements.txt
```

### 6. Run the project

```bash
python main.py
```
